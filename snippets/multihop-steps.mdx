<Steps>
  <Step title="User Derives Secret">
    The user derives a secret using the recoverable mechanism and creates an HTLC for Solver(AB) on Chain A.
  </Step>
  <Step title="Solver(AB) Locks Funds">
    Once detected, Solver(AB) creates an HTLC for Solver(BC) on Chain B using the same hashlock.
  </Step>
  <Step title="Solver(BC) Locks Funds">
    Solver(BC) then creates an HTLC for the user on Chain C using the same hashlock.
  </Step>
  <Step title="Secret Reveal">
    The user reveals the secret on Chain C, enabling all participants to unlock their funds across the chain sequence.
  </Step>
  <Step title="Unlocks">
    Solver(BC) releases the user’s funds on Chain C and claims funds on Chain B. Solver(AB) claims funds on Chain A.
  </Step>
</Steps>
